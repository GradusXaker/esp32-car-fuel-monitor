# Схема подключения ESP32 Car Fuel Monitor

## Компоненты

| Компонент | Модель | Кол-во |
|-----------|--------|--------|
| ESP32 DevKit v1 | ESP-WROOM-32 | 1 |
| OLED дисплей | SSD1306 0.96" I2C | 1 |
| Датчик уровня топлива | Резистивный поплавок | 1 |
| Кнопка | Тактовая 6x6mm | 1 |
| Резистор | 10k Ом | 1 |
| Резистор | 220 Ом | 1 |
| Конденсатор | 100uF 16V | 1 |

## Схема подключения

### ESP32 Pinout

```
                    ┌─────────────────────────┐
                    │      ESP32 DevKit       │
                    │                         │
        3.3V ───────┤1  EN               20  GND├────── GND
        GPIO23 ─────┤3  GPIO23           19  GPIO21├────── SDA (OLED)
        GPIO22 ─────┤5  GPIO22           17  GPIO21├────── SCL (OLED)
        GPIO34 ─────┤7  GPIO34           15  GPIO27├
                    │                         │
        GPIO0  ─────┤9  GPIO0            13  GPIO14├
                    │                         │
        GND  ───────┤11 GND               11  3.3V├────── 3.3V
                    └─────────────────────────┘
```

### Подключение OLED дисплея (SSD1306 0.96" I2C)

```
OLED Display          ESP32
┌─────────┐           
│ VCC     │─────────── 3.3V (пин 11)
│ GND     │─────────── GND (пин 11)
│ SCK     │─────────── GPIO22 (пин 5)
│ SDA     │─────────── GPIO21 (пин 17)
└─────────┘
```

### Подключение датчика топлива (поплавок)

```
Бак автомобиля        ESP32
┌─────────────┐       
│  Поплавок   │       
│     ┌┴┐     │       
│     └┬┘     │       
│      │      │       
│   Сигнал ───┼─────── GPIO34 (пин 7)
│      │      │       
│      └──────┼─────── GND через 10k Ом
│             │       
└─────────────┘       

Примечание: Датчик топлива обычно имеет 3 провода:
- Питание (+12V от автомобиля) - НЕ подключать к ESP32!
- Сигнал - через делитель напряжения к GPIO34
- Земля - общая с ESP32
```

### Делитель напряжения для датчика

```
Автомобиль (12V)
     │
    ┌┴┐
    │ │ R1 10k Ом
    │ │
    └┬┘
     ├─────────── GPIO34 (ADC)
     │
    ┌┴┐
    │ │ R2 10k Ом  
    │ │
    └┬┘
     │
    GND

Важно: ESP32 работает от 3.3V! 
Используйте делитель напряжения для защиты входа ADC.
```

## Таблица подключений

| ESP32 Pin | Компонент | Провод | Примечание |
|-----------|-----------|--------|------------|
| 3.3V | OLED VCC | Красный | Питание дисплея |
| GND | OLED GND | Чёрный | Общая земля |
| GPIO21 | OLED SCL | Жёлтый | I2C Clock |
| GPIO22 | OLED SDA | Зелёный | I2C Data |
| GPIO34 | Датчик | Синий | ADC вход |
| GPIO0 | Кнопка | Белый | Калибровка |
| GPIO2 | LED | Оранжевый | Индикатор |

## Важные замечания

### ⚠️ Предупреждение по напряжению

- ESP32 работает от **3.3V**
- Максимальное напряжение на GPIO: **3.6V**
- Автомобильная сеть: **12V** (до 14.5V при работе двигателя)

**Обязательно используйте делитель напряжения!**

### Формула делителя

```
Vout = Vin × (R2 / (R1 + R2))

Для Vin = 12V, Vout = 3.3V:
R1 = 10k Ом, R2 = 3.3k Ом
Или используйте два 10k для Vout ≈ 6V (безопасно для ESP32)
```

### Защита от помех

```
Добавьте конденсатор 100uF между 3.3V и GND
для фильтрации помех автомобиля.
```

### Монтаж

1. Установите ESP32 в защитный корпус
2. Используйте термоусадку для всех соединений
3. Закрепите провода стяжками
4. Разместите устройство вдали от источников тепла
