## 🔌 Схема подключения

### OLED дисплей (SSD1306)

| OLED | ESP32 |
|------|-------|
| VCC | 3.3V |
| GND | GND |
| SCK | GPIO22 |
| SDA | GPIO21 |

### Датчик топлива

| Датчик | ESP32 |
|--------|-------|
| Сигнал | GPIO34 (через делитель) |
| GND | GND |

⚠️ **Важно:** Используйте делитель напряжения 10k/10k для защиты ESP32!

### Кнопка

| Кнопка | ESP32 |
|--------|-------|
| Один контакт | GPIO0 |
| Другой контакт | GND |

Полная схема в [docs/WIRING_DIAGRAM.md](docs/WIRING_DIAGRAM.md)

## 💾 Установка прошивки

### Требования
- Arduino IDE или PlatformIO
- Драйверы CP210x или CH340 (для USB)
- Библиотеки:
  - Adafruit SSD1306
  - Adafruit GFX

### Шаги

1. **Установите библиотеки в Arduino IDE:**
   ```
   Sketch → Include Library → Manage Libraries
   - Adafruit SSD1306
   - Adafruit GFX Library
   ```

2. **Откройте прошивку:**
   ```
   firmware/CarFuelMonitor.ino
   ```

3. **Настройте параметры:**
   ```cpp
   #define TANK_CAPACITY 60  // Ваша ёмкость бака в литрах
   ```

4. **Загрузите прошивку:**
   - Подключите ESP32 по USB
   - Выберите плату: "DOIT ESP32 DEVKIT V1"
   - Нажмите Upload

## 🔧 Калибровка

### Метод 1: Кнопка на ESP32

1. **Полный бак:**
   - Заполните бак полностью
   - Удерживайте кнопку 3 секунды
   - LED загорится

2. **Пустой бак:**
   - Опустошите бак (или до минимума)
   - Удерживайте кнопку 5 секунд

3. **Сброс:**
   - Удерживайте кнопку 7 секунд

### Метод 2: Android приложение

1. Подключитесь к устройству
2. Нажмите "Калибровать ПОЛНЫЙ бак"
3. Нажмите "Калибровать ПУСТОЙ бак"

---

### Сборка из исходников

### Установка

1. Откройте проект в Android Studio:
   ```
   android/
   ```

2. Синхронизируйте Gradle

3. Соберите APK:
   ```
   Build → Build Bundle(s) / APK(s) → Build APK(s)
   ```

4. Установите на телефон

### Использование

1. **Сопряжение:**
   - Включите Bluetooth на телефоне
   - Сопрягите с "CarFuelMonitor"

2. **Подключение:**
   - Откройте приложение
   - Выберите устройство из списка
   - Нажмите "Подключить"

3. **Мониторинг:**
   - Данные обновляются каждые 2 секунды
   - Прогресс бар показывает уровень топлива

## 📡 Команды Bluetooth

| Команда | Описание |
|---------|----------|
| `GET_STATUS` | Получить все данные |
| `SET_FULL` | Калибровать полный бак |
| `SET_EMPTY` | Калибровать пустой бак |
| `RESET_STATS` | Сбросить статистику |
| `SET_TANK:XX` | Установить ёмкость бака (XX литров) |
| `HELP` | Список команд |

### Формат данных

```json
{
  "fuel_level": 75.5,
  "fuel_liters": 45.3,
  "consumption": 8.5,
  "distance": 150,
  "total_used": 12.7,
  "adc": 3200,
  "tank": 60
}
```

## ❓ FAQ

### Q: Неточные показания?
**A:** Проверьте калибровку. Убедитесь, что делитель напряжения правильно подключен.

### Q: Не подключается Bluetooth?
**A:** Убедитесь, что устройство сопряжено в настройках телефона перед подключением в приложении.

### Q: Как увеличить точность?
**A:** Используйте больше сэмплов для усреднения (измените AVG_SAMPLES в прошивке).

### Q: Можно ли использовать с GPS?
**A:** Да! Подключите GPS модуль и замените симуляцию расстояния на реальные данные.

### Q: Как изменить интервал обновления?
**A:** Измените UPDATE_INTERVAL в прошивке (в миллисекундах).
