# Инструкция по установке

## 🔧 Прошивка ESP32

### 1. Установка Arduino IDE
Скачайте с официального сайта: https://www.arduino.cc/en/software

### 2. Установка платы ESP32
1. Откройте **File → Preferences**
2. В поле **"Additional Boards Manager URLs"** вставьте:
   ```
   https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
   ```
3. Откройте **Tools → Board → Boards Manager**
4. Найдите **"esp32"** и установите

### 3. Установка библиотек
Откройте **Sketch → Include Library → Manage Libraries** и установите:

| Библиотека | Версия | Автор |
|------------|--------|-------|
| **Adafruit SSD1306** | 2.5.x | Adafruit |
| **Adafruit GFX Library** | 1.11.x | Adafruit |

### 4. Настройка платы
```
Tools → Board → ESP32 Dev Module
Tools → Upload Speed → 115200
Tools → Port → /dev/ttyUSB0 (Linux) или COM3 (Windows)
```

### 5. Загрузка прошивки
1. Откройте `firmware/CarFuelMonitor.ino`
2. Нажмите **Upload** (стрелка вправо)
3. Дождитесь завершения

## 📱 Установка Android приложения

### Вариант 1: Готовый APK
1. Скачайте `CarFuelMonitor-v2.2.0-premium.apk`
2. Разрешите установку из неизвестных источников
3. Установите APK

### Вариант 2: Сборка из исходников
```bash
cd android
./gradlew assembleDebug
```
APK будет в: `app/build/outputs/apk/debug/app-debug.apk`

## 🔌 Подключение компонентов

### OLED дисплей (SSD1306 I2C)
| OLED | ESP32 |
|------|-------|
| VCC | 3.3V |
| GND | GND |
| SCL | GPIO22 |
| SDA | GPIO21 |

### Датчик топлива
| Датчик | ESP32 |
|--------|-------|
| Сигнал | GPIO34 |
| GND | GND (через 10k Ом) |

### Кнопка
| Кнопка | ESP32 |
|--------|-------|
| Один контакт | GPIO0 |
| Другой контакт | GND |

## ⚙️ Настройка

### Калибровка полного бака
1. Заполните бак полностью
2. Удерживайте кнопку 1 секунду
3. LED загорится

### Калибровка пустого бака
1. Опустошите бак (до минимума)
2. Удерживайте кнопку 5 секунд

### Калибровка полубака
1. Установите уровень топлива около 50%
2. Удерживайте кнопку 3 секунды

### Сброс статистики
1. Удерживайте кнопку 7 секунд

## 📱 Использование Android приложения

1. Включите Bluetooth на телефоне
2. Сопрягите с устройством **"CarFuelMonitor"**
3. Откройте приложение
4. Выберите устройство из списка
5. Нажмите **"Подключить"**

## ❓ Решение проблем

### Ошибка "board not found"
- Установите драйверы CP210x или CH340
- Проверьте USB кабель

### Ошибка библиотек
- Убедитесь, что установлены **обе** библиотеки Adafruit
- Перезапустите Arduino IDE

### OLED не работает
- Проверьте подключение SDA/SCL
- Попробуйте адрес `0x3D` вместо `0x3C`

### Bluetooth не подключается
- Удалите устройство из сопряжённых
- Сопрягите заново
